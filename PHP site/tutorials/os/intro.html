INTRODUCTION

An operating system is a system software which provides an interface between user and computer hardware .

It mainly has two functions :

1.Manages computer hardware

2.Provides environment for running other softwares.

HISTORY

In 1940's came the first generation :Those computers dont have operating system and were so primitive that programs were often entered one bit at a time on rows of mechanical switches(plug borad). Programing languages were unknown at this time. Also employed vaccum tubes for calculations.

in 1950's came second generation computers :The general motors research Laboratories implemented first os for IBM701.The routine improved with the introduction of punch cards although one can execute only job at a time.These computers were called SINGLE-STREAM BATCH PROCESSING SYSTEMS because program and data were submitted in groups or batches.

in 1960's came third generation computers : They were also batch processing system but with the advantage of multiprogramming. using concept of spooling and time slicing processor keeps on switching form one job to another and hence multiple job can be executed concurrently.

Then came Fourth generation : With the progress of MicroProcessor technologies.LSI(large scale integration) techniques helped devise the personal computer as powerfull as main frame computer of 1970's.Now this systems consist of Integrated circuits and had operating system for workstations and personal computers.Two operating system dominated :MsDOS by MICROSOFT for INTEL8088 cpu and its sucessors & Unix os for MOTOROLA 6899 CPU family.
